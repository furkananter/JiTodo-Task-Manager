## Projenin Çalıştırılması

Bu projeyi çalıştırmak için aşağıdaki adımları takip edin.

1) Projeyi klonladıktan sonra;
```bash
  npm install
```
2) Localhost'ta çalıştırmak için;
```bash
  npm start
```
3) TailwindCSS'i çalıştırmak için;
```bash
  npm run watch
```

2) For running on Localhost;

```bash
  npm run watch
```
